## Quick start

1. To install server dependencies run:

```shell
$ yarn install
```

and for client:

```shell
$ yarn install:client
```

2. Copy `.env.template` file to `.env.development` and `.env.production` and make the necessary changes there.

3. Run server and client in development mode.

```shell
$ yarn dev
```
